### 1. Backend (Spring Boot)

  ```bash
  mvn spring-boot:run
  ```
- Backend runs on `http://localhost:8080`

### 2. Frontend (React)
- Install Node.js 18+
- In the `src` folder, run:
  ```bash
  npm install
  npm start
  ```
- Frontend runs on `http://localhost:8081`
